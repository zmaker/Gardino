These firmwares are useful to test basic boards functionality

Master Board
------------
1. gardino_nrf_sd - test if NRF is working along with SD card reader (both SPI)
2. gardino_nrf_sd_RTC - test if NRF is working along with SD card reader and RTC clock (SPI + I2C)
3. gardino_nrf_sd_RTC_rele - test if NRF is working
4. MKRWIFI_NRF24 - NRF24 RTX test

Satellite Board
---------------
1. RX_nano - test if NFR24 is receiving data 
1. TX_nano - test if NFR24 can transmit data
